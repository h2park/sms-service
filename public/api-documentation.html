Send Message

Send an SMS
Supports multiple recipients using a delimiter ‘<’

Request

POST /message

Body Parameters

dst required
string

The destination numbers, The numbers should include valid telephone numbers with the country and area code. e.g. 1602855255 For multiple numbers include the '<' delimiter.

text required
string

The contents of the SMS message. If the contents are greater than 160 characters, they will be broken up into multiple messages.

Response

Status: 200 OK
